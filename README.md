# 🌱 FarmToFork – Database & Inventory Management System

## 🚀 About the Project

FarmToFork is a full database-driven application designed to manage the agricultural supply chain — connecting farmers, warehouses, and clients in a structured and efficient way.

The system handles inventory, purchasing, sales transactions, and analytical reporting, ensuring real-time stock updates and data consistency.

---

## 💡 Key Features

* 📦 Inventory management with real-time updates
* 🧾 Purchase & sales transaction processing
* ⚠️ Low stock and expiry tracking
* 📊 20+ analytical SQL reports
* 🔄 Full CRUD operations for all entities
* 🔒 Data validation and integrity enforcement

---

## 🧱 Tech Stack

* **Java (JavaFX)** – User Interface
* **MySQL** – Database
* **JDBC** – Database Connectivity

---

## 🏗️ System Design

The application follows a clean layered architecture:

* **UI Layer** (JavaFX)
* **DAO Layer** (Data Access Objects)
* **Database Layer** (MySQL)

This structure improves scalability, maintainability, and separation of concerns.
